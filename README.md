### 서비스 대표 기능
- 영화 추천 알고리즘  
  : 문답을 통해 추천하는 장르의 영화 목록을 보여주는 서비스   
  사용자의 당일 기분/취향에 따라 맞춤형으로 추천 목록 제공

- 추천 영화 목록 제공   
  : TMDB 에서 제공하는 인기순, 최신순, 탑 랭크 순 영화 목록 제공 서비스   
  사용자 개인이 아닌 대중들에게 인기 있는 목록 제공
---

### 영화 추천 알고리즘
1. 3가지 질문을 통해 사용자의 현재 기분을 추측
2. 답변의 분위기와 어울리는 영화 장르를 추출
3. TMDB에서 제공하는 영화 목록 중 2에서 추출한 장르에 해당하는 영화들을 랜덤으로 추천
